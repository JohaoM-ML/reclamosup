# Visual Reference — Universidad del Pacífico (ReclamoUP)

**Date:** 2026-05-30  
**Sources:** Blackboard login + Portal DAAAR intranet  
**Target:** ReclamoUP institutional UX

## 1. Core Aesthetic

**Style name:** Corporate Academic Portal — UP Institutional

**Philosophy:** Confianza institucional mediante contraste alto (azul marino + acento naranja en login; azul marino + azul acción en portal), tipografía sans-serif limpia y layout de tres columnas con sidebar fijo.

**Influences:** Blackboard UP login, portal DAAAR Pregrado, dashboards gubernamentales/educativos peruanos.

## 4. Key Design Elements

- `navy-sidebar-fixed` — sidebar 260px, full height, white icons/text
- `login-hero-overlay` — full-viewport navy gradient over subtle texture
- `orange-cta-login` — single orange button, full width, rounded 6px
- `action-blue-buttons` — rectangular blue CTAs in dashboard
- `card-soft-shadow` — white cards, 1px border #E5E7EB, shadow-sm
- `table-institutional` — gray header row, white body, hairline dividers
- `active-nav-pill` — left border or bg highlight in action blue
- `uppercase-form-labels` — login labels in white caps

## 5. Visual Concept

Portal académico que transmite oficialidad UP: el login es ceremonial (overlay oscuro, CTA naranja único); el dashboard es operativo (sidebar navy, contenido blanco, acciones en azul). ReclamoUP adopta la paleta sin copiar assets institucionales — tipografía Geist mantiene modernidad.
